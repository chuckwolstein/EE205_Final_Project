***IMPORTANT!!!  The game runs at different speeds on different systems.

-In Windows subsystem for Linux (Ubuntu Developer Mode), speeds of moving
cars are much slower.  We recommend you go into include/Macros-and-libs.hpp
and edit the macro MAX_SPEED accordingly.

Recommended settings for Windows Ubuntu:  15 to 20

Recommended settings for Linux (VirtualBox) or Mac Terminal:  4 to 7




COMPILING AND RUNNING
The command to compile and run the program (it's an all-in-one command) 
depends on your environment.


1.  Linux (virtualBox) users:
make

2.  Apple computers using terminal:
make mac
